ch.Tabs

Description

Tabs lets you create tabs for static and dynamic content.

How-to

{% highlight javascript %} // Create a new Tabs. var tabs = new ch.Tabs(el); {% endhighlight %}

Parameters

  • el - HTMLElement : A HTMLElement to create an instance of ch.Tabs.
  • options - Object : Options to customize an instance.

Extends

  • ch.Component

Properties

.Component#name String

The name of a component.

.container HTMLElement

The tabs container.

.panel HTMLElement

The container of tab panels.

.tabpanels Array

A collection of tab panel.

.Tabs#name String

The name of the component.

{% highlight javascript %} // You can reach the associated instance. var tabs = $(selector).data('tabs'); {% endhighlight %}

.triggers HTMLElement

The tabs triggers.

.uid Number

A unique id to identify the instance of a component.
