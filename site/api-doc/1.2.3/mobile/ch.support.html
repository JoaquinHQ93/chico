support (Namespace)

Description

Returns a data object with features supported by the device

Properties

.fx Boolean

Checks if the $ library has fx methods.

{% highlight javascript %} if (ch.support.fx) { // Some code here! } {% endhighlight %}

.touch Boolean

Checks if the User Agent support touch events.

{% highlight javascript %} if (ch.support.touch) { // Some code here! } {% endhighlight %}

.transition Boolean

Verify that CSS Transitions are supported (or any of its browser-specific implementations).

{% highlight javascript %} if (ch.support.transition) { // Some code here! } {% endhighlight %}
