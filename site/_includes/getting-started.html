Getting started

How to use Chico in your site.

Overview

Chico supports mobile and desktop —called “ui”— platforms and has two major releases: v1x and v2x.

To get started, please, select version and platform:

  • v1x
    • Mobile
    • UI
  • v2x
    • Mobile
    • UI

Resources

To get started you first need to get a copy of Chico. You have 2 options:

Install it using npm:

{% highlight sh %} npm install chico {% endhighlight %}

Use the resources from our CDN:

{% highlight text %} // HTTP/1.1 https://ui.mlstatic.com/chico/{{ chico_current_version }}/{{ chico_current_platform }}/chico.min.css https://ui.mlstatic.com/chico/{{ chico_current_version }}/{{ chico_current_platform }}/chico.min.js // HTTP/2 https://http2.mlstatic.com/ui/chico/{{ chico_current_version }}/{{ chico_current_platform }}/chico.min.css https://http2.mlstatic.com/ui/chico/{{ chico_current_version }}/{{ chico_current_platform }}/chico.min.js {% endhighlight %}

For a specific version, please check the releases in the repository.
